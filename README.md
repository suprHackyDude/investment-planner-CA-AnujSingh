# Personalized Investment Planner

Static single-page app for Instagram lead generation.

## GitHub Pages

After this repo is on GitHub:

1. Open the repository on GitHub → **Settings** → **Pages** (left sidebar).
2. Under **Build and deployment** → **Source**, choose **Deploy from a branch**.
3. **Branch:** `main` (or `master`), folder **`/ (root)`**, then **Save**.
4. Your site will be available at:

   `https://<your-username>.github.io/<repository-name>/`

   (GitHub shows the exact URL on the Pages settings screen after the first deploy; wait 1–2 minutes.)

## Local preview

python3 -m http.server 8765
